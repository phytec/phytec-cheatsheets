Users and permissions
*********************

.. warning::

    Linux permissions is a very complex topic. This cheat sheet only covers the basics.

Overview
========

In Linux each to file and directory an owner, a group (that the owner belongs to), 
and permissions is assigned. 
A user can be assigned to multiple groups and can own multiple files.
